# 🚀 MLM (Multi-Level Marketing) System - Java + MySQL

## 📌 Project Overview
This project is a console-based Multi-Level Marketing (MLM) system built using Java and MySQL.  
It handles user registration, referral-based tree structure, wallet system, and commission distribution up to 4 levels.

---

## ✨ Features

### 👤 User Features
- User Registration with Referral Code
- Login System
- Wallet Balance View
- Deposit Request System
- Withdrawal Request System
- Admin Panel for User Management
- Referral Tree (Upline & Downline)
- Transaction History

### 🧑‍💼 Admin Features
- Admin Login System
- View All Users
- Block / Unblock Users
- Approve / Reject Deposits & Withdrawals
- Commission Plan Setup (4 Levels)
- Commission Reports (User-wise & Level-wise)
- Wallet Management System
- View Full User Tree

---

## 🧠 Core MLM Logic
- Multi-level referral structure (Upline/Downline)
- Commission distribution up to 4 levels
- Deposit-based commission system
- Referral-based commission system
- Wallet auto update system
- Complete transaction tracking
